Regular Expressions

April 9, 2016

Today, we are going to talk about regular expressions. In the world of web development, when it comes to complex data with huge of string structures, you need to know your fair share of regular expressions, regular expressions are ways to manipulate strings in order to get a desired output. For example in Ruby:

 puts "Max is so sad today!".sub('sad','happy')

 Ouput:   "Max is so happy today!"

As you can see we replaced the word sad with happy, However this is not great example of regular expressions. They are usually inside a pair of forward slash "//" like so: 

"Hello string".scan(/./) {|s| puts s }

The code above prints every character of the string, the "/./" stands for every character, you also noticed the method scan which is an enurable that is used with strings. it does exaclty what its name suggests. Now let's print every two characters, as you may alreay guessed, we will use "/../" for every two characters.

The previous illustration prints everyting including spaces because they are accounted as characters also, so if we want to be handy, we can print the string nicely with no spaces.

"Hello string".scan(/\w\w/) {|s| puts s }

The "/w" is a special character, it means any alphanumeric or underscore, there are many other special characthers such as "\d" for any digit, "\D" anything that is not a digit, "\s" for white space etc. Here is an example:

"My birthday is on april 12th".scan(/\d/) do |d| puts d end

Output:

We can also use the match operator "=~" with regular expressions, let's say we want to print only the vowels in a given string. whe can do this:

"Please take my vowels".scan(/./) {|c| puts c if c =~ /[aeiou]/}

The scan method will pass every character to the code block then they are checked by the match operator to see if they are vowels "/[aeiou]/". We can even do this a nicer way the with the match method.

"Please take my vowels".scan(/./) {|c| puts c if c.match(/[aeiou]/)}

Ouput:

Lastly, Ranges can also be used to extract a specific target range.

"Ruby is fun!".each_char {|d| puts d if d =~ /[a-f]/}

Output:
b
f

This was an introduction to Regular expressions, there is so much more about them that it is a branch of computer science, you will become a more successull web developer if you could get a solid foundaton on regular expressions.
